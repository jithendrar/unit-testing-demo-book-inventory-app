# unit-testing-demo-book-inventory-app

Prerequisites:

- NodeJS
- npm
- Mongo (https://docs.mongodb.com/manual/installation/)
- Bower
    npm install -g bower
- Karma CLI
    npm install -g karma-cli

To run application:

- Install dependencies
    npm install

- Install bower libraries
    bower install

- Have a local instance of mongo db running
    mongod

- Run Server
    node server/server

To run unit tests:
    karma start
